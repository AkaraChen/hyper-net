# Hyper Net

Just a toy project.

## Features

1. Based on [net/http](https://pkg.go.dev/net/http)
2. Easy to use
3. Support for middleware (onion architecture)
4. Support for `http.Handler`(By calling `hyper.HTTPHandlerIntoHyper`)
